# 📊 MLOps – Ciclo de vida de modelos de Machine Learning

Este proyecto trata sobre **MLOps (Machine Learning Operations)**, una disciplina que explica cómo se gestionan los modelos de Machine Learning cuando pasan a un entorno real de producción.

A lo largo de la presentación se explica el ciclo completo de un modelo: desde la obtención y preparación de datos, el entrenamiento y evaluación, hasta su despliegue, monitorización y reentrenamiento automático.

También se muestran las principales ideas de automatización en MLOps y herramientas utilizadas en entornos reales como CI/CD, Git, Airflow, MLflow y Kubeflow.
